# MLB Award Predictor — Streamlit App

## Deployment to Streamlit Cloud

### Step 1: Upload your model file
After running `mlb_award_prediction_v3.ipynb`, download `mlb_award_models.pkl` from your Colab drive.

### Step 2: Push to GitHub
```bash
git init
git add app.py requirements.txt README.md
git commit -m "MLB award predictor"
git remote add origin https://github.com/YOUR_USER/mlb-award-predictor.git
git push -u origin main
```

### Step 3: Deploy on Streamlit Cloud
1. Go to [share.streamlit.io](https://share.streamlit.io)
2. Connect your GitHub repo
3. Set **Main file path** to `app.py`
4. Click **Deploy**

### Step 4: Upload the model
Once deployed, use the **sidebar uploader** on the live app to upload `mlb_award_models.pkl`.
The app runs in demo mode (heuristic) until the model file is present.

## Running Locally
```bash
pip install -r requirements.txt
# Place mlb_award_models.pkl in this folder
streamlit run app.py
```

## Model File Format
The app expects `mlb_award_models.pkl` (generated by the notebook) with:
```python
{
    'models': TUNED_MODELS,   # dict: award -> {model: TwoStageAwardModel, val_spearman: float}
    'all_data': all_data,     # dict: award -> {feats: list, prep: sklearn Pipeline, tr/va/te tuples}
}
```
